Step 1 — Decision Gating Agent

Purpose

Interpret uploaded material and identify the real decision.

Input

What it does

extracts supporting decisions
separates operational text from strategic text

Output

Required outputs

Where the Pipeline Is Strong

These elements are already aligned:

Decision-first workflow
MIOS → BAOS → signals sequence
External signal separation
Deterministic judgment core
Structured response step

The logic is coherent.

Where More Detail Was Needed (Now Added)

These clarifications prevent future drift:

MIOS and BAOS produce signals only
External Scout collects context only
Judge produces probability only
Analysis explains only
Respond recommends only
Simulation projects only

Each agent has a single job.

The Most Important Structural Rule

Every agent must obey this:

One responsibility per agent.

If an agent performs multiple roles, the system becomes unstable.
