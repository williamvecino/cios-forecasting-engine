- number of resolved cases
- active/inactive correction

Goal:
Make system maturity visible and prevent overclaiming.

4. TRACE MODE ENRICHMENT
Add to trace mode:
- question type
- therapy area
- case mode

Goal:
Allow the analyst to know not only why the forecast is what it is, but how much to trust the calibration context around it.
