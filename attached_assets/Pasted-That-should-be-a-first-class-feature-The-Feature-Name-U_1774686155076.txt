The Feature Name

Use language that matches how work actually arrives.

Best options:

Recommended default:

Import Project

Short. Familiar. Works across pharma and agencies.

What This Feature Must Do

When someone sends:

an email
an RFP
a market summary
meeting notes
a PDF
a spreadsheet
a slide deck

The system should:

Read the material
Identify the decision question
Extract signals
Suggest missing signals
Prepare the case

Automatically.

The User Flow

Very simple.

STEP 1

User clicks:

Import Project

STEP 2

User uploads:

PDF
Email text
Excel
Slides
Document

STEP 3

System responds:

We found the following in your project.

Decision question
Will Viatris launch the generic aripiprazole vial in 2026 or be delayed?

Key signals detected

No manufacturing readiness timeline announced
Competitor long-acting formulation approved
Payer coverage not yet defined

Missing signals to investigate

Process validation status
Commercial production scheduling
FDA inspection readiness

Then:

Proceed to Judgment

That is the behavior you want.

What to Tell Replit

Add an “Import Project” feature as a primary entry point. Users should be able to upload documents such as PDFs, emails, slides, or spreadsheets. The system must automatically interpret the materials, extract the decision question, identify key signals, suggest missing signals, and prepare the case for judgment without requiring manual setup.
