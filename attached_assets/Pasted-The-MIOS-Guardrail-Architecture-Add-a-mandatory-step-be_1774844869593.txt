Implementation — Minimal Python Validator

This is production-ready logic.

import requests

def verify_pmid(pmid):
    url = f"https://pubmed.ncbi.nlm.nih.gov/{pmid}/"
    response = requests.get(url)

    if response.status_code == 200:
        return "VALID"
    else:
        return "NOT FOUND"

def verify_doi(doi):
    url = f"https://api.crossref.org/works/{doi}"
    response = requests.get(url)

    if response.status_code == 200:
        return "VALID"
    else:
        return "NOT FOUND"
MIOS Rule to Store (recommended)

Identifier Integrity Rule

The system must not generate PMIDs, DOIs, or registry numbers under any circumstance.
All identifiers must be extracted from source documents and verified against external registries before acceptance.
