What to Tell Replit — Core Instruction

Use this exactly.

Replace the current structured question-entry process with a natural language question interpreter. Users should be able to type any question in plain language. The system must automatically extract the decision type, event, outcomes, time horizon, and likely constraints, then present the interpreted version for confirmation before proceeding. The goal is to minimize user steps and make question entry feel conversational and intuitive.

Rule 4
